Guia de Comandos MySQL - Manutenção do Banco de Dados

Banco de Dados: railway | Host: junction.proxy.rlwy.net:34797 | Usuário: root

1. 🔌 Conexão e Seleção do Banco

ComandoDescrição
USE railway;Selecionar o banco de dados
SELECT DATABASE();Ver qual banco está em uso

2. 📋 Estrutura do Banco

ComandoDescrição
SHOW TABLES;Listar todas as tabelas
DESCRIBE nome_tabela;Ver estrutura da tabela
SHOW CREATE TABLE nome_tabela;Ver detalhes completos da tabela

3. 🔍 Consultar Dados (SELECT)

ComandoDescrição
SELECT * FROM funcionarios;Ver todos os funcionários
SELECT id, nome_completo, email FROM funcionarios;Ver apenas alguns campos
SELECT * FROM cardapio LIMIT 10;Ver com limite de registros
SELECT * FROM funcionarios ORDER BY nome_completo ASC;Ordenar por nome (A-Z)
SELECT * FROM funcionarios ORDER BY id DESC LIMIT 5;Ver os 5 últimos cadastrados

4. 🎯 Filtrar Dados (WHERE)

ComandoDescrição
SELECT * FROM funcionarios WHERE cargo = 'Gerente';Filtrar por cargo
SELECT * FROM cardapio WHERE categoria = 'hamburguer' AND disponivel = 1;Múltiplas condições
SELECT * FROM funcionarios WHERE nome_completo LIKE '%Silva%';Busca parcial (contém Silva)
SELECT * FROM pedidos WHERE status = 'aberto';Pedidos em aberto

5. 📊 Contar Registros

ComandoDescrição
SELECT COUNT(*) FROM funcionarios;Total de funcionários
SELECT cargo, COUNT(*) as total FROM funcionarios GROUP BY cargo;Total por cargo
SELECT COUNT(*) FROM cardapio WHERE disponivel = 1;Itens disponíveis no cardápio
SELECT categoria, COUNT(*) FROM cardapio GROUP BY categoria;Total por categoria

6. ✏️ Inserir Dados (INSERT)

ComandoDescrição
INSERT INTO funcionarios (nome_completo, cpf, email, senha, cargo, salario, turno) VALUES ('João Silva', '123.456.789-00', 'joao@email.com', '123456', 'Atendente', 2500.00, 'tarde');Inserir novo funcionário
INSERT INTO cardapio (nome, categoria, preco, descricao, disponivel) VALUES ('X-Burger Especial', 'hamburguer', 32.90, 'Pão, carne, queijo, bacon', 1);Inserir novo item no cardápio
INSERT INTO fornecedores (cnpj, razao_social, telefone, email) VALUES ('12.345.678/0001-90', 'Distribuidora ABC', '(11) 99999-9999', 'contato@abc.com.br');Inserir novo fornecedor

7. ✏️ Atualizar Dados (UPDATE)

ComandoDescrição
UPDATE cardapio SET preco = 25.90 WHERE id = 5;Atualizar preço de um item
UPDATE cardapio SET disponivel = 0, motivo_indisponibilidade = 'Fora de estoque' WHERE id = 37;Desativar item do cardápio
UPDATE funcionarios SET salario = 3000.00 WHERE id = 4;Atualizar salário
UPDATE funcionarios SET ativo = 0 WHERE id = 2;Desativar funcionário
UPDATE pedidos SET status = 'entregue' WHERE id = 10;Atualizar status do pedido

8. ❌ Deletar Dados (DELETE)

⚠️ ATENÇÃO! Sempre use WHERE no DELETE. Teste com SELECT primeiro!
ComandoDescrição
DELETE FROM cardapio WHERE id = 99;Deletar um item do cardápio
DELETE FROM funcionarios WHERE id = 10;Deletar um funcionário

9. 🔧 Alterar Estrutura da Tabela (ALTER)

ComandoDescrição
ALTER TABLE funcionarios ADD COLUMN telefone_fixo VARCHAR(15);Adicionar uma coluna
ALTER TABLE funcionarios DROP COLUMN telefone_fixo;Remover uma coluna
ALTER TABLE funcionarios MODIFY COLUMN telefone_celular VARCHAR(20);Modificar tipo da coluna
ALTER TABLE funcionarios RENAME COLUMN telefone TO telefone_contato;Renomear coluna

10. 📈 Relatórios Rápidos

ComandoDescrição
SELECT data_pedido, COUNT(*) as pedidos, SUM(total) as faturamento FROM pedidos GROUP BY data_pedido ORDER BY data_pedido DESC LIMIT 10;Total de vendas por dia
SELECT nome, motivo_indisponibilidade FROM cardapio WHERE disponivel = 0;Itens indisponíveis
SELECT nome_completo, cargo, salario FROM funcionarios WHERE ativo = 1;Funcionários ativos
SELECT item, quantidade, minimo FROM estoque WHERE quantidade <= minimo;Produtos com estoque crítico
SELECT razao_social, telefone, email FROM fornecedores WHERE ativo = 1;Fornecedores ativos

11. 📌 Resumo Rápido

ComandoFunção
SHOW TABLES;Lista tabelas
DESCRIBE nome_tabela;Mostra estrutura
SELECT * FROM tabela;Ver todos dados
INSERT INTO tabela VALUES(...);Inserir registro
UPDATE tabela SET campo=valor WHERE id=X;Atualizar registro
DELETE FROM tabela WHERE id=X;Deletar registro
SELECT COUNT(*) FROM tabela;Contar registros
SELECT * FROM tabela LIMIT 10;Limitar resultados
SELECT * FROM tabela ORDER BY campo;Ordenar resultados
SELECT * FROM tabela WHERE condicao;Filtrar resultados
⚠️ DICAS DE SEGURANÇA!

• Sempre use WHERE no UPDATE/DELETE - Evita alterar/deletar todos os registros
• Teste com SELECT antes do DELETE - Verifica quais registros serão afetados
• Use LIMIT em consultas grandes - Evita travamentos
• Faça backup antes de alterações grandes - Permite recuperar dados
