# Hamburgueria-Na-Chapa# 🍔 Hamburgueria Na Chapa

Sistema completo de gestão para hamburgueria, desenvolvido com foco em controle de pedidos, funcionários, estoque e administração.

---

## 🚀 Tecnologias Utilizadas

- PHP 8+
- MySQL
- HTML5
- CSS3
- JavaScript (Fetch API)
- Apache
- Docker (deploy)
- Railway (deploy em produção)

---

## 📦 Funcionalidades

### 👤 Autenticação
- Login de funcionários
- Controle de nível de acesso:
  - Administrador
  - Estoquista
  - Funcionário

---

### 🧾 Gestão de Pedidos
- Criar pedidos
- Visualizar pedidos
- Controle de status

---

### 📊 Administração
- Controle de funcionários
- Controle de permissões
- Logs de ações do sistema

---

### 📦 Estoque
- Controle de produtos
- Atualização de ingredientes
- Monitoramento de itens disponíveis

---

### 🔐 Sistema de Permissões
- Controle de acesso por nível de usuário
- Ocultação de menus no frontend
- Proteção de áreas administrativas

---

## 🗄️ Banco de Dados

O sistema utiliza MySQL com tabelas principais:

- funcionarios
- pedidos
- produtos
- estoque
- logs_sistema
